Project information

  • Category: Web development
  • Client: Brew run
  • App URL: app.brewrun.co.nz/
  • Project Description: A web-based solution to assist customers in selling their products, specifically coffee beans, online. The solution allows consumers to place orders online, and the system automatically assigns a driver for delivery. The solution also includes an admin console to manage various aspects of the system, including products, accounts, and drivers.
  • Technologies: LaravelAngular

My responsibilities

  • Developed the web solution using AngularJS as the frontend framework and Laravel as the backend.
  • Focused primarily on developing APIs and implementing features for the admin console on the backend.
  • Conducted code reviews and merged source code changes to ensure the quality and maintainability of the codebase.
  • Provided deployment support by configuring the Nginx web server.

Insights & Gains

  • Implemented a specialized and reusable canisters system to significantly reduce packaging waste. Initial purchases incur a nominal $3 fee for the canister, which can be returned for refills without any extra charge.
  • Developed a range of subscription plans, providing customers with the flexibility to tailor their delivery frequency according to their coffee consumption. They have the option to select weekly, bi-weekly, or monthly deliveries, ensuring a steady supply of their favorite coffee blends.
