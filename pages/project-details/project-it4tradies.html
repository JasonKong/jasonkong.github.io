Project information

  • Category: Web development
  • Client: IT4Tradies
  • Project Description: The application serves as a middleware between Simpro and Jotform, enabling users to search for Simpro job numbers and auto-populating related details in Jotform. Once the user submits the form in Jotform, a webhook is triggered, facilitating the download of PDFs and attachments, which are then uploaded to specific folders in Simpro. This setup streamlines data transfer and integration between the two platforms.
  • Technologies: LaravelVue
