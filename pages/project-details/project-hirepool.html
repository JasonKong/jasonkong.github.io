Project information

  • Category: Web development
  • Client: Hirepool
  • Project URL: www.hirepool.co.nz
  • Project Description: A web application for one of the biggest equipment hire companies in New Zealand. The application includes an internal system that allows staff to create contracts from scratch by searching for products, checking availability, pricing, and other relevant information. In addition, it also includes a public website that allows customers to place orders online.
  • Technologies: Laravel

  • Supported for the deployment process.
